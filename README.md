# VoiceBroadcast
商户收款语音播报 Merchant collection voice broadcast

该项目包含如下功能：                                                                          
    1. 播报商户收款的具体金额，精确到小数点后两位                                                              
    2. 支持顺序播报，直接调用方法即可，开发者不用任何特殊处理

该功能已运行在数十万Android设备上，趋近于成熟稳定，欢迎issues
